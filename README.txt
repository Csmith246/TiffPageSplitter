This batch file splits .tif files up by pages.

Directions: Put .tif you want to split in same directory as tiffSplitter.bat. Then run the batch file.

Note: You need ImageMagick downloaded and added to window's PATH variables in order for this to work. The download .exe you run should add magick to the path, but if not here is some help: 
https://helpdeskgeek.com/windows-10/add-windows-path-environment-variable/


ImageMagick download:
https://www.imagemagick.org/script/index.php
